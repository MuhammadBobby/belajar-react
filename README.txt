Babel
--> Babel adalah sebuah dictionary yang digunakan untuk menerjemahkan jsx (seperti <h1>) menjadi Javascript
    jika kita ingin mengimplementasikan elemen jsx ke dalam react, maka gunakan babel untuk menerjemahkan nya
    --> contoh di latihan 1 / index.html

jsx
--> ekstensi sintaks untuk JS yang dapat mendeskripsikan UI dengan sintaks seperti html
    3 aturan JSX :
        a. harus selalu mengembalikan 1 elemen
        b. harus selalu tutup tag nya
        c. gunakan camelCase untuk atribut
